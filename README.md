This is my first project, which focuses on developing a Student Record System. Such systems are widely implemented in schools and colleges to efficiently store and manage student information.
In this project, I have utilized three key data fields as input: name, roll number, and marks. However, the system can be easily extended to accommodate additional data fields as required.
It basically asks the user for input, then writes it into a file,, and finally shows the details.
## How to Run
1. Install a C++ compiler (g++)
2. Compile: g++ fileio.cpp -o student_record
3. Run: ./student_record
